SuaveIO

TraceHeader

A record that keeps track of what request this is. In an uint64 there are 18 446 744 073 709 551 616 number of possible values, so you can be fairly certain a given request id is unique, given a good random number generator.

mk(traceId spanParentId)
Signature: (traceId:uint64 option) -> (spanParentId:uint64 option) -> TraceHeader

Create a new TraceHeader with the given trace_id and span_parent_id. This generates a new id and places it in trace_id AND req_id if no trace_id parameter is supplied. Unless span_parent_id is given, that field is defaulted to None, as suave cannot know the "origin span", so to speak.
