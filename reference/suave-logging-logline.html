Suave.IO

LogLine

When logging, write a log line like this with the source of your log line as well as a message and an optional exception.

Record Fields

Record FieldDescription
exception
Signature: exn option

an optional exception
level
Signature: LogLevel

the level that this log line has
message
Signature: string

the message that the application wants to log
path
Signature: string

the source of the log line, e.g. 'ModuleName.FunctionName'
trace
Signature: TraceHeader

the trace id and span id If using tracing, then this LogLine is an annotation to a span instead of a 'pure' log entry
ts_utc_ticks
Signature: int64

timestamp when this log line was created
